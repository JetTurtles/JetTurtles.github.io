欢迎来到游戏开发知识天地！

这是一个专属的游戏开发学习空间，记录我在游戏开发中的点点滴滴。

TypeScript 基础

TypeScript 是一种由微软开发的自由和开源的编程语言。它是 JavaScript 的一个严格超集，并添加了可选的静态类型和基于类的面向对象编程。

Cocos Creator 基础

Cocos Creator 既是一款高效、轻量开源的跨平台图形引擎，也是一个实时 3D 内容创作平台。 Cocos Creator 能完美胜任 2D、3D 的游戏开发工作。

Blender 基础

Blender 是一款免费开源的 3D 创作套件。掌握 Blender 基础，可帮助开发者为游戏打造精美的 3D 模型、场景，提升游戏视觉效果，是游戏美术资源制作的重要技能 。
